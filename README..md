A machine learning project to classify news articles as *Real* or *Fake* using natural language processing techniques

# Problem Statement
The spread of fake news can mislead users and impact public trust.  
This project helps automatically detect fake news using text-based features.

# Tools & Techniques
- Python
- Pandas, NumPy
- Scikit-learn
- NLTK
- TF-IDF Vectorization
- Logistic Regression

# Key Outcomes
- Built a text classification model for fake news detection  
- Applied text preprocessing and feature extraction  
- Achieved reliable performance on unseen news data
